TableGeneratorForTeX

・作成者　ぽむぽむ

・更新日　2018年10月31日

・使い方

　(1) basファイルをダウンロード

　(2) TeX形式の表を作成したいExcelファイルを開く

　(3) 開発タブからVisualBasicを開く

　(4) ウィンドウ上部のファイルタブから、「ファイルのインポート」を開く

　(5) (1)でダウンロードしたbasファイルを開く

　(6) 表にしたい領域を選択する

　(7) 開発タブのマクロから「TableGeneratorForTeX」を選択して実行する

　(8) コードが生成されるので、Ctrl+Cなどでコピーする
